## How to Use

### Method 1: Double-Click (Easiest)

**Windows:**
1. Double-click `start.bat`
2. Browser opens automatically
3. Done! 🎉

**Mac/Linux:**
1. Right-click `start.sh` → select "Open"
2. OR run: `./start.sh`
3. Browser opens automatically
4. Done! 🎉

### Method 2: Command Line

```bash
cd /path/to/LU-Reviewer-App
python3 server.py
```
Then visit: **http://localhost:8000**

### Method 3: Any Web Server

Copy the entire folder to any web server (Apache, Nginx, etc.)
- Just serve it as-is, no build required
- Works with GitHub Pages
- Works with web hosting
