## Need Help?

**App won't start?**
- Make sure Python 3 is installed: `python3 --version`
- Try a different port: `python3 -m http.server 9000`

**Browser won't open?**
- Manually go to: http://localhost:8000

**Permission denied?** (Mac/Linux)
- Run: `chmod +x start.sh`
