## If Something Goes Wrong

### Problem: No output after clicking RUN
**Solution**: 
- Check your code for syntax errors
- Make sure you have `return 0;` at end of main
- Click the ERROR message to see what went wrong

### Problem: "Undefined reference" error
**Solution**:
- Add `#include <stdio.h>` for printf/scanf
- Add `#include <math.h>` for math functions
- Add `#include <string.h>` for string functions

### Problem: scanf not accepting input
**Solution**:
- Put your input in the Input box FIRST
- Then click RUN
- Each scanf needs a value

### Problem: Very slow execution
**Solution**:
- This is normal for complex programs in JavaScript
- Most programs should run in under 1 second
- Really complex programs might take 2-5 seconds
