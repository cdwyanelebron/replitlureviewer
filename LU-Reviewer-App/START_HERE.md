## Troubleshooting

### "Port 8000 already in use"
```bash
python3 -m http.server 9000
# Visit: http://localhost:9000
```

### "Python command not found"
- Install Python: https://python.org
- OR use any web server (Apache, Nginx, etc.)

### "Permission denied" (Mac/Linux)
```bash
chmod +x start.sh
./start.sh
```
